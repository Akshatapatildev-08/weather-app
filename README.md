# Weatherstack Glass SaaS App

## Setup

```bash
npm install
npm run dev
```

Optional API key override:

```bash
cp .env.example .env
```

## Features

- Current weather
- Forecast weather (with min temp + condition filters)
- Historical weather by date
- Marine weather by date
- Location lookup with autocomplete
- Search UX with instant suggestions
- Glassmorphic SaaS dashboard UI
